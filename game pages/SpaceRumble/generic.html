Space Rumble
Unreal, full game

VR Space combat simulator with 3 modes: MOBA, CTF, ring collection.

Design Document

No Code - all scripts were written using visual scripting which uses binary files. in addition git space ran out(complicated). The game suffers from sever fps drop since translating visual scripting into c++ during run time is very costly and since project is very corrupted in order to rewrite the important tick event(physics, cooldowns, etc...) it needs to be written agian in a black project. transfering files transfer the corruption and since most scripts will be made into half visual scripts and half regular ones which is also a great chance to add multiplier. Another benefit will be to use an older version of the engine, one that works with the getnamo python servers so Deep Learning could be implemented.

The player controls a ship from third person view and activates its abilities by putting his hand inside one of the boxes portrayed in the demonstration video and using either the grip or trigget according to the desired effect of the ability. the game has 4 different custom physics for the spaceships so one could choose between more realistic space movement to one more resembling flying in the atmosphere.
