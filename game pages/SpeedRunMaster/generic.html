Speed Run Master
Unity game

Vie to complete each track even faster

Code Download game

A driving game where you need to complete the track using abilities you aquire randomly by collecting pickup boxes. When the player completes the track he will see a ghost of his previous run driving along side him. Abilities: Rockets that destroy obstacles, jump, hook, and shield that destroys obstacles. Keys:
Gas - gamepad~down arrow, right trigger. keyboard~w,s/up and down arrows.
Turning - gamepad~left joystick. keyboard~a,d/left and right arrows.
Camera - gamepad~right joystick. keyboard~mouse movement.
Pause menu - gamepad~start. keyboard~esc.
menu back - gamepad~button east.
Activate pickup - gamepad~left trigger. keyboard~space bar.
Switch pickup - gamepad~left shoulder. keyboard~tab.
Break - gamepad~button south. keyboard~ctrl.
